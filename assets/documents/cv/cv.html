Projects

Planning App

C# | Software Development

Intended to be a one-size-fits-all kind of situation, I wanted to make a planning app that's as close to perfect as I can manage. I've always been a very detailed orientated person and have found that the apps that I've tried to use to plan out certain things never quite have everything I need. So, instead of waiting for the app I would love to see get developed by someone else, I decided to make it.

The intention is to have enough features and flexibility that the user can create whatever list they desire without the app seeming too overwhelming to the average person. Examples of this could include: a to-do style shopping list that includes item prices, a timetable that shows you your set daily tasks, or even a flow chart style to-do list so that you can plan each step chronologically.

Initially I was planning to make this in Android Studio but complications led me to using Unity instead for my first software development project.

This project is still currently under development.
